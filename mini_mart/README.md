# Mini Mart web-app

a mini mart web reacct app built using react & tailwindcss by using external dumyjson free api, it provides products, category, search system
dark/light theme & responsive ui

### Features
- provides a total list of products with info (title, description, price, rating, etc.)
- search bar, category select (filter products), theme toggle
- Responsive ui, clean file structure.

### Tech Stack
- Frontend: React.js, TailwindCSS, Vite, Npm
- Backend / DB: dumyjson api (an external api integration for products)
- Deployment: Vercel (frontend)

### Live: https://ecommerce-site-smoky.vercel.app
