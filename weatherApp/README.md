# Weather Web-App

Weather Web-app frontend focused built using react & tailwindcss for responsive & reusable components, it fetches weather status based on input 
search & provides past 7 days & next 3 days weather status.

## Features
- search and get instant weather status with past 7 days & next forecast 3 days
- search bar suggestions.
- Responsive UI for all devices.
- dark/light theme.

## Tech stack
- React, TailwindCSS, Vite, Npm Free Weather API. 

Live: https://weather-app-opal-kappa-13.vercel.app
