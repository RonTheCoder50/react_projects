## Calculator APP

a calculator app built using react & tailwindcss, that provides arithmetic bodmas calculation clean for any type of calculations with 
responsive ui & clean/moduler layout (buttons, color)

### Features
- Arithmetic bodmas calculation.
- Responsive & Clean Layout.
- theme toggle

### Tech Stack
- React, TailwindCSS, Vite, Npm, Math.js (api).
